# Playbook And Platform Sources

## Pinned Growth Playbook Basis

The audit method is grounded in the following Growth Playbook pages at commit [`b2ae914`](https://github.com/krillinai/growth-playbook/commit/b2ae914). Pinning keeps the conceptual basis reviewable as the Playbook evolves.

- [Andrew Foxwell](https://github.com/krillinai/growth-playbook/blob/b2ae914/case-library/people/andrew-foxwell/README.md): practitioner source for Meta advertising, creative strategy, measurement discipline, and operating context. Use as a methodological lens, not as the Skill identity or a substitute for account evidence.
- [Acquisition](https://github.com/krillinai/growth-playbook/blob/b2ae914/handbook/acquisition/README.md): channel fit, paid acquisition, message continuity, customer quality, and downstream growth constraints.
- [Metrics](https://github.com/krillinai/growth-playbook/blob/b2ae914/handbook/metrics/README.md): metric definitions, qualified outcomes, metric trees, cohorts, guardrails, and decision use.
- [Experimentation](https://github.com/krillinai/growth-playbook/blob/b2ae914/handbook/experimentation/README.md): causal questions, experiment design, guardrails, interpretation, and decision rules.
- [Growth Infrastructure](https://github.com/krillinai/growth-playbook/blob/b2ae914/handbook/growth-infrastructure/README.md): event quality, identity, data contracts, attribution, observability, and reliable operating systems.

Conceptual sources shape questions and methods. They do not verify a private account, current platform behavior, economic threshold, or causal result.

## Current Meta Implementation References

Consult current official documentation before implementing an authorized API or measurement integration. These pages are time-sensitive and were cataloged on 2026-07-23:

- [Marketing API Insights](https://developers.facebook.com/docs/marketing-api/insights/)
- [Ad Account Insights reference](https://developers.facebook.com/docs/marketing-api/reference/ad-account/insights/)
- [Ad Creative reference](https://developers.facebook.com/docs/marketing-api/reference/ad-creative/)
- [Conversions API](https://developers.facebook.com/docs/marketing-api/conversions-api/)
- [Marketing API authorization](https://developers.facebook.com/docs/marketing-api/overview/authorization/)
- [Graph API changelog](https://developers.facebook.com/docs/graph-api/changelog/)

Official implementation documentation defines supported fields, permissions, versions, and integration behavior. It does not supply account performance, incrementality, customer quality, or business economics. This Skill does not perform the integrations described by those references.

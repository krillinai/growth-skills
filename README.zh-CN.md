# Growth Skills by clawee.ai

[English](README.md) | [简体中文](README.zh-CN.md)

一个面向增长与营销工作的开放技能集合。Growth Skills 将营销知识沉淀为可复用的工作流，帮助从业者和 AI Agent 覆盖并衔接增长全生命周期中的关键任务。

## 覆盖领域

- **内容创作** - 调研、选题、写作、编辑与内容分发。
- **数据洞察** - 指标衡量、数据分析、实验与决策支持。
- **广告投放** - 活动规划、创意策略、媒体采买与投放优化。
- **获客转化** - 用户增长、线索获取、激活与转化率提升。
- **营销策划** - 市场定位、上市策略、营销活动设计与增长规划。

## 增长全生命周期

这个集合致力于支持从首次触达到持续增长的连贯工作流：

**认知 → 获客 → 激活 → 转化 → 留存 → 扩张**

**Discover → Acquire → Activate → Convert → Retain → Expand**

## 持续完善

本项目仍处于早期阶段。随着更多实用工作流得到开发和验证，我们会持续收录新的技能。
